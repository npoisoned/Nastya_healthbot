# Nastya_healthbot

Проект №2 — Telegram-бот для расчёта нормы воды и калорий, а также трекинга еды и тренировок.

Автор: Хвостовец Анастасия

## Команды
- /set_profile — заполнение профиля (вес, рост, возраст, активность, город, цель калорий)
- /log_water <мл> — добавить выпитую воду
- /log_food <продукт> — добавить еду (калорийность берётся из OpenFoodFacts, при необходимости можно ввести вручную)
- /log_workout <тип> <мин> — добавить тренировку (расход калорий + доп. вода)
- /check_progress — показать прогресс за сегодня

## Технологии
- Python, aiogram
- OpenWeatherMap (погода)
- OpenFoodFacts (калории продуктов)

## Деплой
Проект развёрнут на Render как Background Worker. Токены и ключи хранятся в переменных окружения.

## Скриншоты
Скриншоты работы бота и логи деплоя находятся в папке `screens/`.
